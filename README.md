#  Snake Game – Proyecto Interactivo en JavaScript

Proyecto desarrollado con HTML, CSS y JavaScript que recrea el clásico juego **Snake**, pero con funciones mejoradas, interfaz moderna y un panel de control avanzado.  
Incluye selección de tamaño del tablero, velocidad, skins, modos de juego, audio dinámico y sistema de puntaje con récord guardado.

---

##  Características Principales

Interfaz responsive y moderna  
Control de velocidad del juego  
Selector de tamaño del tablero (10x10 – 20x20)  
Modos de juego:
- Paredes (muerte en colisión)
- Wrap (teletransporte por bordes)

Skins visuales de la serpiente:
- Clásico  
- Neón  
- Retro  
- Matrix  

Sonidos dinámicos (comer, pausa, game over)  
Registro del mejor puntaje usando `localStorage`  
Temporizador en tiempo real  
Sistema de niveles y velocidad progresiva  
Controles intuitivos mediante teclado  
Efectos visuales llamativos y colores personalizados

---

##  Funcionamiento del Juego

El jugador controla una serpiente en un tablero.  
El objetivo es comer la comida que aparece aleatoriamente, haciendo que la serpiente crezca.

Si se activa el modo **paredes**, la serpiente muere al chocar contra un borde.  
En modo **wrap**, puede atravesar los bordes y aparecer en el lado opuesto.

La velocidad aumenta automáticamente conforme suben los puntos.
